1. Context: What is HTML?

HTML (HyperText Markup Language) is the language used to structure a web page.

A typical HTML page looks like this:

<html>
   <head>
       <!-- page information -->
   </head>

   <body>
       <!-- visible content -->
   </body>
</html>

Important:

  • Head → information about the page (title, metadata, CSS…)
  • Body → the content the user sees (text, images, links, etc.)

Everything you studied in the notes is mainly about elements used inside the <body>.

2. Headings (Titles)

HTML provides 6 levels of titles.

They are used to organize content like chapters in a book.

Syntax

<h1>Title</h1>
<h2>Title</h2>
<h3>Title</h3>
<h4>Title</h4>
<h5>Title</h5>
<h6>Title</h6>

Example

<h1>Main Title</h1>
<h2>Section</h2>
<h3>Subsection</h3>

Result:

  • <h1> → biggest title
  • <h6> → smallest title

Example:

<h1>Adel</h1>
<h2>Mohamed</h2>

3. Horizontal Line <hr>

This tag creates a horizontal line to separate content.

Syntax

<hr>

Example:

<h1>Chapter 1</h1>
<hr>
<p>Text of the chapter</p>

Result: a line separating sections.

4. Block vs Inline Elements

HTML elements behave in two main ways.

This is related to the display property in CSS.

4.1 Inline Elements

Inline elements stay on the same line.

They only take the space they need.

Example elements:

<span>
<img>
<a>

Example:

This is <span>inline text</span> example

Result: everything stays on one line.

4.2 Block Elements

Block elements start on a new line.

They usually take the full width.

Examples:

<p>
<div>
<h1>
<hr>

Example:

<p>Paragraph 1</p>
<p>Paragraph 2</p>

Each paragraph appears on a new line.

5. Line Break <br>

This tag creates a new line inside text.

Syntax

<br>

Example:

Hello<br>
World

Result:

Hello
World

6. Image Element <img>

The <img> tag is used to display images.

Important:

img is a self-closing tag.

Basic Syntax

<img src="image.jpg">

6.1 Important Attributes of <img>

1. src

Defines the path of the image.

Example:

<img src="photo.jpg">

2. width and height

Define image size.

Example:

<img src="photo.jpg" width="200" height="150">

3. alt

Alternative text if the image cannot load.

Example:

<img src="photo.jpg" alt="profile picture">

7. Links <a>

Links allow users to navigate to another page or website.

Syntax

<a href="URL">Text</a>

Example:

<a href="http://www.google.com">Google</a>

8. Opening Links in New Tab

You can open a link in a new tab using target.

<a href="http://google.com" target="_blank">Google</a>

9. Link Title Attribute

The title attribute shows a tooltip.

Example:

<a href="http://google.com" title="Go to Google">Google</a>

When you hover the mouse → text appears.

10. Using an Image as a Link

You can place an image inside a link.

Example:

<a href="http://google.com">
   <img src="logo.png">
</a>

When the user clicks the image → it opens Google.

11. Internal Links (Inside the Same Page)

You can link to a specific section of the page.

Step 1 — Create an ID

<img id="tim" src="tim.jpg">

Step 2 — Create the link

<a href="#tim">Go to image</a>

# means jump to the element with that ID.

1. Unordered List (UL)

An unordered list shows items using bullets instead of numbers.

It is used when order does not matter.

1.1 Basic Syntax

A list uses two elements:

  • <ul> → the list container
  • <li> → list item

Example:

<ul>
   <li>Java</li>
   <li>PHP</li>
   <li>CSS</li>
   <li>C#</li>
   <li>JavaScript</li>
</ul>

2. Changing Bullet Style

HTML allows different bullet styles using the type attribute.

Example:

<ul type="disc">
<ul type="square">
<ul type="circle">

Possible bullet styles:

typeresult
disc●
circle○
square■

3. Ordered List (OL)

An ordered list shows items with numbers or letters.

It is used when order matters.

5.1 Basic Syntax

<ol>
   <li>Java</li>
   <li>C#</li>
   <li>PHP</li>
   <li>JavaScript</li>
</ol>

4. Ordered List Types

The numbering style can change using the type attribute.

Example:

<ol type="1">   numbers
<ol type="A">   capital letters
<ol type="a">   small letters
<ol type="I">   roman numerals
<ol type="i">   small roman numerals

5. Starting Number

You can choose where numbering starts.

Example:

<ol start="50">
   <li>Java</li>
   <li>PHP</li>
</ol>

6. Reverse Order

You can reverse numbering using:

<ol reversed>

Example:

<ol reversed>
   <li>Java</li>
   <li>PHP</li>
   <li>CSS</li>
</ol>

1. Paragraph <p>

A paragraph is the most basic text element.

Syntax

<p>This is a paragraph</p>

Each <p> starts on a new line.

2. Bold Text

There are two ways to make text bold.

1. <b>

Used only for visual bold style.

<b>Important</b>

2. <strong>

Used for important text (semantic meaning).

<strong>Important text</strong>

Browsers display both as bold, but <strong> means the text is important.

3. Smaller Text

Use:

<small>text</small>

Example:

<p>This is normal text</p>
<small>This is smaller text</small>

4. Italic Text

Two tags exist.

1. <i>

Just makes text italic.

<i>italic text</i>

2. <em>

Means emphasized text.

<em>important emphasis</em>

Browsers usually display it as italic.

5. Quotation <blockquote>

Used for long quotations.

Example:

<blockquote>
HTML is the structure of the web.
</blockquote>

Browsers usually indent the quote.

6. Short Quote <q>

Used for short quotes inside text.

Example:

<p>He said <q>HTML is easy</q></p>

Result:

He said “HTML is easy”.

7. Highlighted Text <mark>

Used to highlight text.

Example:

<mark>Important word</mark>

Browsers show it with yellow background.

8. Underlined Text <u>

Example:

<u>This word is underlined</u>

9. Deleted Text <del>

Shows text that has been removed.

Example:

<del>This is wrong</del>

Result:

This is wrong

10. Inserted Text <ins>

Shows newly inserted text.

Example:

<ins>This is new</ins>

Usually appears underlined.

11. Superscript <sup>

Used for powers in mathematics.

Example:

5<sup>2</sup>

Result:

5²

12. Subscript <sub>

Used for chemical formulas.

Example:

H<sub>2</sub>O

Result:

H₂O

13. Non-Breaking Space &nbsp;

Normally HTML collapses spaces.

Example:

Hello      World

HTML will display:

Hello World

If you want extra space, use:

&nbsp;

Example:

Hello&nbsp;&nbsp;&nbsp;World

14. HTML Entities

Some characters cannot be typed directly in HTML.

They must use HTML entities.

An entity always starts with:

&

and ends with:

;

Common HTML Entities

SymbolEntity
<&lt;
>&gt;
&&amp;
©&copy;
®&reg;
€&euro;

Example:

<p>&copy; 2026 My Website</p>
